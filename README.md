
# System Information Monitor

Created with node.js, this tool scans your system and provides specifications about your machine. This includes general information, CPU data, and memory configuration.


## Dependencies
express, systeminformation

## How to use
Simply clone the repository and install the required dependencies. Once the project is initialized, navigate to the root directory and run the start script "node server.js" to start the application. View the application by visiting http://localhost:3000 (if you keep the default port 3000).
